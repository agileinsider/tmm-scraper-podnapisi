# Podnapisi Subtitle Scraper for tinyMediaManager

A subtitles scraper for [tinyMediaManager](https://github.com/tinyMediaManager) which uses [podnapisi]
(https://podnapisi.net), which increases the chances of finding subtitles for eastern european 
titles.

## installation

You just need to drop the jar file into the tinyMediaManager plugins directory and restart tinyMediaManager.  You 
will then have a new subtitle scraper available.
